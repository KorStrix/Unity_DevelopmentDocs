# 채팅방

## 공지사항


### 채팅방 규칙

- 상호간에 존칭 사용, 비방 금지.
- 5분이상 사담은 잡담방에서.

### 권장사항

- 답변이 질문에 도움이 되셨다면 감사인사 부탁.
- 질문을 해도 답변을 안할 경우 자신의 질문이 답변하기 어려운 것인지 체크해보기.
